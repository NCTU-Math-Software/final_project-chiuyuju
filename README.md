**射擊遊戲**

`如何開始遊戲`
1. 下載shooting_game.m
2. 執行shooting_game.m，依照畫面指示操作。

`簡介`
* 開始遊戲後，將在畫面上隨機出現目標物，使用者需以滑鼠左鍵點擊目標物。
* 若使用者失誤超過規定次數，則遊戲結束。
